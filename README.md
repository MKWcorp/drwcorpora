# DRW Corpora - Landing Page

Landing page profesional untuk DRW Corpora - perusahaan konsultasi bisnis dan transformasi digital.

## Teknologi

- **React 18** - UI library
- **Vite** - Build tool dan dev server
- **TypeScript** - Type safety
- **Lucide React** - Professional icons
- **CSS3** - Styling dan animations

## Struktur Project

```
client/
├── index.html          # Entry point HTML
├── src/
│   ├── main.tsx        # React entry point
│   ├── App.tsx         # Main component
│   ├── index.css       # Global styles
│   ├── components/     # React components
│   │   ├── Navbar.tsx
│   │   ├── HeroSection.tsx
│   │   ├── AboutSection.tsx
│   │   ├── VisionMissionSection.tsx
│   │   ├── ServicesSection.tsx
│   │   ├── GallerySection.tsx
│   │   └── ContactFooter.tsx
│   └── styles/         # Component styles
│       ├── navbar.css
│       ├── hero.css
│       ├── about.css
│       ├── vision.css
│       ├── services.css
│       ├── gallery.css
│       └── contact.css
```

## Instalasi & Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
